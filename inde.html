Create a new repository on the command line:


echo "# git-commits" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin url
git push -u origin main
git clone url
git status
git pull
git updates
cd ..

Push an existing repository from the command line:


git remote add origin url
git branch -M main
git push -u origin
